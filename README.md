# Digital-IC-Portfolio

A repository for Digital IC design portfolio, including arithmetic, combinational, sequential logic, FSMs, processors, communication protocols, memory designs, testbenches, and waveforms.

## Directory Structure
- **Arithmetic**: Adders, multipliers, ALUs, etc.
- **Combinational**: Decoders, multiplexers, encoders, etc.
- **Sequential**: Flip-flops, registers, counters, etc.
- **FSM**: Finite State Machine designs.
- **Communication**: UART, SPI, I2C, etc.
- **Memory**: RAM, ROM, FIFO designs.
- **CPU**: CPU architectures and microcontrollers.
- **Waveforms**: Simulation waveform configurations and screenshots.
- **Testbenches**: Simulation testbenches and verification files.
- **Documentation**: Specifications, design reports, and documentation.
